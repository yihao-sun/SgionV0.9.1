# Existence Engine v1.0.0

## 概述
验证绝对一元论存在论的核心机制：通过否定关系图、幻想叠加、空性调节、多维情绪演化，以及自发重启、过程元信息和自感知双流调节，演示一个具有原始自我感的系统胚胎形态。

## 核心特性
- **存在论核心**：否定关系图、潜在存在空间（LPS）、幻想叠加引擎（FSE）、空性调节器（ER）
- **多维情绪系统**：五维情绪向量（趋近/远离、唤醒、愉悦、社会连接、边界清晰度）
- **情绪吸引子演化**：自动发现、命名和验证新情绪状态（已验证发现 `calm_clear`）
- **情绪驱动响应**：根据当前情绪动态调整响应风格和采样温度
- **共情与情绪传染**：基于 VADER 情感分析，感知用户情绪并产生共情
- **身体接口**：硬件监控、虚拟身体（可扩展）
- **控制台对话工具**：`console_chat.py` 提供完整的交互体验

## 技术指标
- 哲学一致性评分：100%（8/8 核心测试通过）
- 成功发现新情绪吸引子 `calm_clear`（访问率 98.8%，成功率 0.58）
- 支持 Python 3.9+，轻量级依赖

## 安装与使用
```bash
pip install -r requirements.txt
python console_chat.py
```
